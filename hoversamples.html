• Expanding Circle

    One of my favorite hover effects. When the text box is hovered a circle grows and fills the box with color. 

    
                        .cs--circle--card__wrapper:hover 
                        .cs--circle--hover--effect {
                        transform: scale(12);
                        opacity: .8;
                        }

  • Double Circle Variant

    A variation of the previous hover effect. This one takes advantage of two circles to create a ripple effect.

    
                        .circle--card__wrapper:hover 
                        .first--circle--hover--effect, 
                        .circle--card__wrapper:hover 
                        .second--circle--hover--effect {
                        transform: scale(12);
                        opacity: .8;
                        }

  • Hover Reveal

    When the card is hovered the content is revealed.

    
                        .icon--float--wrapper {
                        transform: translateY(85%);
                        transition: all .7s ease-in-out;
                        }
                        .icon--card__wrapper:hover 
                        .icon--float--wrapper {
                        transform: translateY(0);
                        }
